# Life Timeline

A web application for visualizing and managing your life timeline. Create, track, and reflect on important events, milestones, and moments that shape your life story.

## Features

- **Event Creation**: Easily add events to your timeline with dates, titles, and descriptions
- **Interactive Timeline**: View your events on an interactive, scrollable timeline
- **Event Management**: Edit or delete events as needed
- **Date-based Organization**: Events are automatically sorted chronologically
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
- **Data Persistence**: Your timeline is saved locally in your browser

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or database setup required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/jamthedude/Timeline.git
cd Timeline
```

2. Open the application:
```bash
# Simply open index.html in your web browser
# Or use a local development server:
python -m http.server 8000
# Then navigate to http://localhost:8000
```

## Usage

### Adding an Event

1. Click the "Add Event" button
2. Enter the event date
3. Add a title for your event
4. (Optional) Add a description with more details
5. Click "Save" to add the event to your timeline

### Viewing Your Timeline

- Scroll through your timeline to see all events in chronological order
- Click on an event to view full details
- Events are displayed with dates, titles, and descriptions

### Editing or Deleting Events

- Click on an event to select it
- Use the "Edit" button to modify event details
- Use the "Delete" button to remove an event
- Confirm your action when prompted
